## Контекст

Создание системы документации для проекта FEONT на основе проверенной системы из проекта FEMSQ, адаптированной под специфику работы с графом знаний. Проект FEONT - это исследовательский инструмент для моделирования и эволюции бизнес-процессов через граф знаний на основе RDF/SHACL технологий.

**Цель:** Создать полную систему документации, включающую базовые JSON-файлы, правила ведения, структуру для планов/резюме чатов и предварительных документов.
